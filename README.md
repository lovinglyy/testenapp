#### Requisitos
- [Node.js](https://nodejs.org/en/ "Node.js")
- npm ou yarn

#### Instalação
```bash
    git clone https://github.com/lovinglyy/testenapp
	cd ~/testenapp
	npm install

```    
#### Como rodar
```bash
cd <pasta do projeto>
npm start
```
O servidor irá rodar em modo de desenvolvimento, não produção.
